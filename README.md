# Hotel Lakshya Sheesh Mahal Indore

A luxury hotel website built with React, Vite, and Tailwind CSS.

## Features

- Responsive navbar with mobile menu
- Full-screen hero section
- About, Facilities, Gallery, and Contact sections
- Footer with newsletter signup
- Floating WhatsApp button
- Navy blue, gold, and white premium theme
- Scroll animations and hover effects

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production

```bash
npm run build
npm run preview
```
